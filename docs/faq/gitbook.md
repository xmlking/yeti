### How to embed Hints?

{% hint style='info' %}
Tips: "styles" translate to the different hint types : info, success, danger & warning.
Checkout https://docs.gitbook.com/editing-content/embeds
{% endhint %}

```markdown
{% hint style='info' %}
Important info: this note needs to be highlighted
{% endhint %}
```

{% endhint %}

{% endhint %}

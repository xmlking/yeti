# pnpm

## Commands

```bash
npm install -g pnpm
# Upgrading
pnpm add -g pnpm

pnpm add debug -W
pnpm add debug --global

pnpm up
pnpm up --latest
pnpm ls prune
pnpm ls
pnpm audit
pnpm why
```

### create Nx workspace

```bash
pnpx create-nx-workspace@latest --package-manager pnpm
```
